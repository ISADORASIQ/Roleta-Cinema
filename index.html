APIs Utilizadas:

API de Dados de Filmes (YTS API):

URL da API: https://yts.mx/api/v2/movie_details.json?movie_id={movie_id}

Descrição: Esta API fornece informações detalhadas sobre filmes, incluindo duração, gênero, classificação, sinopse e capa do filme. Permite buscar informações de filmes com base no ID do filme.

API de Tradução (Google Translate API):

URL da API: https://translation.googleapis.com/language/translate/v2?key={sua_chave_de_api}&source={idioma_origem}&target={idioma_destino}&q={texto_a_ser_traduzido}

Descrição: Esta API é usada para traduzir texto de um idioma de origem para um idioma de destino. No site, ela é usada para traduzir a sinopse dos filmes para tornar o conteúdo acessível a um público mais amplo.
